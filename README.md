# 🐍 Atividade Python 8 | Programação Orientada a Objetos (POO)

Este repositório contém uma atividade prática desenvolvida durante o curso de **Python**, com foco em **Programação Orientada a Objetos (POO)**, um dos pilares da programação moderna.

O objetivo desta atividade é compreender como estruturar códigos utilizando **classes e objetos**, organizando melhor a lógica do programa e tornando o código mais reutilizável e escalável.

---

## 🧠 Conteúdos abordados

O notebook trabalha os seguintes conceitos:

1. **Criação de Classes**

   * Definição de classes com `class`
   * Estrutura básica de uma classe

2. **Método Construtor (`__init__`)**

   * Inicialização de atributos
   * Definição de propriedades do objeto

3. **Atributos e Métodos**

   * Atributos de instância
   * Métodos personalizados
   * Uso do `self`

4. **Instanciação de Objetos**

   * Criação de objetos a partir da classe
   * Acesso a atributos e métodos

5. **Encapsulamento (quando aplicável)**

   * Organização interna da classe
   * Boas práticas de estruturação

6. **Aplicação prática**

   * Criação de exemplos simulando situações reais
   * Organização da lógica de forma orientada a objetos
